Endpoint assumptions

Backend routes currently used

This admin uses the same core backend route structure as your earlier file: /api/health, /api/meta/knowledgebase, /api/admin/submissions, /api/assessment/{id}/status, and /api/admin/submissions/{id}/resend-email. CSV export and local notes/overrides work without backend changes.
